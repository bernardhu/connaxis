# Code of Conduct

## Our Standards

We want this project to be a respectful and effective collaboration space.

Examples of expected behavior:

- Be respectful and professional
- Focus on technical issues and constructive feedback
- Assume good intent, but be precise about problems and impact
- Accept feedback and correct mistakes

Examples of unacceptable behavior:

- Harassment, intimidation, or personal attacks
- Discriminatory language or conduct
- Trolling, baiting, or repeated non-constructive disruption
- Publishing private information without permission

## Scope

This Code of Conduct applies to:

- Project discussions (issues, pull requests, discussions)
- Review comments and commit messages
- Community spaces directly related to this project

## Reporting

If you experience or witness unacceptable behavior:

- Contact the repository maintainer (`@bernardhu`) directly via GitHub
- Avoid posting sensitive personal details in public issues

Reports will be reviewed in good faith and handled as confidentially as practical.

## Enforcement

Maintainers may remove comments, close discussions, or restrict participation for behavior that violates this Code of Conduct.

## Project-Specific Notes

Technical disagreement is expected. Strong critique of code is acceptable; personal attacks are not.
